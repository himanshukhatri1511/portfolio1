My Projects

Advanced Driver Assistance System (ADAS) Implementation

Developed an Advanced Driver Assistance System (ADAS) prototype during the Electronics Circuits Lab project, integrating Raspberry Pi for enhanced functionality. The project focused on real-time object detection and driver fatigue detection to enhance driving safety.

Technologies Used: Arduino, Raspberry Pi, Ultrasonic Sensors, Python, Machine Learning

Key Features:

  • Object Detection using ultrasonic sensor
  • Driver Fatigue Detection with camera system and machine learning
  • Alert System with buzzer

Reflection: This project allowed me to integrate hardware and software components to create a practical safety system for vehicles. It enhanced my understanding of sensor integration, real-time data processing, and machine learning applications in embedded systems.

Automatic Plant Watering System Without Arduino

Developed an Automatic Plant Watering System as part of the Analog Circuits Lab project to automate watering based on soil moisture levels. This system addresses the need for efficient water usage in agriculture and landscaping.

Technologies Used: 555 Timer IC, LM358 Op-Amp IC, Soil Moisture Sensors, Analog Circuit Components

Key Features:

  • Soil Moisture Monitoring with sensors
  • Automated Watering based on moisture levels
  • Component Integration with transistors, resistors, capacitors, potentiometers, LEDs, and relay modules

Reflection: This project provided hands-on experience in analog circuit design and implementation. It allowed me to create a practical and sustainable solution for automated plant watering, enhancing my skills in sensor integration and circuit design.
